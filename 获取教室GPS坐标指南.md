# 获取教室GPS坐标 - 快速指南

## 🎯 目标
获取教室的GPS坐标，用于演示导航功能

## 📍 方法一：使用APP自动获取（最简单）

1. **打开导航页面**
2. **点击"📍 开始定位"按钮**
3. **授权定位权限**
4. **查看控制台日志**（HBuilderX的调试窗口）
5. **找到类似这样的日志：**
   ```
   GPS定位成功: {latitude: 39.9500, longitude: 116.3500, ...}
   ```
6. **记录这个坐标**（latitude和longitude）

## 📍 方法二：使用高德地图APP

1. 打开高德地图APP
2. 搜索你的学校名称
3. 找到教室所在的教学楼
4. 长按教室位置
5. 查看弹出的坐标信息
6. 记录纬度和经度

## 📍 方法三：使用网页工具

1. 打开：https://lbs.amap.com/tools/picker
2. 搜索你的学校
3. 点击教室位置
4. 查看右侧显示的坐标
5. 记录纬度和经度

## ⚙️ 配置坐标范围

获取到教室中心坐标后（例如：纬度39.95，经度116.35），在 `uni_app/pages/navigation/index.vue` 中配置：

```javascript
mapBounds: {
    minLat: 39.9485,  // 中心纬度 - 0.0015（约167米）
    maxLat: 39.9515,  // 中心纬度 + 0.0015（约167米）
    minLng: 116.3485, // 中心经度 - 0.0015（约167米）
    maxLng: 116.3515  // 中心经度 + 0.0015（约167米）
}
```

### 范围说明：
- **0.001度 ≈ 111米**
- **0.0015度 ≈ 167米**
- **0.002度 ≈ 222米**

### 建议范围：
- **教室演示**：0.0015度（约300米x300米）- 方便在教室内外移动
- **小范围测试**：0.001度（约200米x200米）
- **大范围测试**：0.002度（约400米x400米）

## 🎬 演示流程

1. **配置坐标**：根据教室位置设置 mapBounds
2. **测试定位**：在教室里点击"开始定位"，验证是否能获取GPS
3. **演示移动**：
   - 拿着手机在教室里走动
   - 观察地图上的红色圆点实时移动
   - 展示路径和距离实时更新

## 💡 提示

- 如果GPS信号不好（室内），可以使用"手动选择位置"功能
- 演示时可以边走边点击地图，模拟真实移动
- 确保手机和电脑在同一WiFi，方便查看控制台日志

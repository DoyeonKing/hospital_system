# 3.3 医生端功能测试用例

## 一、个人资料管理功能测试用例

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| DOC-INFO-001 | 编辑个人资料 | 1. 登录医生端系统 2. 在医生工作台点击"编辑资料"按钮 3. 在编辑对话框中修改手机号、擅长领域、个人简介等信息 4. 点击"保存"按钮 | 系统成功更新个人资料信息，显示保存成功提示，修改后的信息在个人信息区域正确显示 | | |
| DOC-INFO-002 | 修改密码 | 1. 在医生工作台点击"修改密码"按钮 2. 在对话框中输入原密码和新密码 3. 确认新密码 4. 点击"确认修改"按钮 | 系统成功修改密码，显示修改成功提示，使用新密码可以正常登录 | | |

## 二、患者管理功能测试用例

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| DOC-PATIENT-001 | 查询患者历史就诊记录 | 1. 进入"患者管理"页面 2. 在"查询患者历史就诊记录"输入框中输入患者姓名 3. 点击"查询历史记录"按钮 | 系统显示该患者在该科室的所有历史就诊记录列表，包含就诊日期、诊断、处方等信息 | | |
| DOC-PATIENT-002 | 查看今日排班患者列表 | 1. 进入"患者管理"页面 2. 查看"今日排班信息"区域的"上午排班"和"下午排班"表格 | 表格正确显示今日的患者排班数据，包括序号、姓名、患者类型、手机号、签到时间、预约状态、过敏史、基础病史等信息 | | |
| DOC-PATIENT-003 | 搜索今日排班患者 | 1. 在"今日排班信息"区域的搜索框中输入患者姓名或手机号 2. 点击搜索或按回车键 | 排班表格中只显示匹配搜索条件的患者记录，其他患者记录被过滤 | | |
| DOC-PATIENT-004 | 查看不同日期的排班 | 1. 在"今日排班信息"区域点击日期选择器 2. 选择一个有排班数据的日期 3. 观察排班表格 | 排班表格显示所选日期的患者排班数据，上午和下午排班分别显示 | | |
| DOC-PATIENT-005 | 刷新排班信息 | 1. 在"患者管理"页面查看当前排班信息 2. 点击"刷新"按钮 | 排班信息重新加载，显示最新的患者排班数据 | | |

## 三、我的排班功能测试用例

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| DOC-SCHEDULE-001 | 查看周视图排班 | 1. 进入"我的排班"页面 2. 默认显示"周视图"标签页 3. 观察排班网格显示 | 页面显示当前周的排班信息，以网格形式展示，包含日期、时间段、诊室等信息，已排班的时段显示排班卡片 | | |
| DOC-SCHEDULE-002 | 查看月视图排班 | 1. 在"我的排班"页面 2. 点击"月视图"标签页 3. 观察排班显示 | 页面切换到月视图，显示整月的排班概览，可以看到一个月内所有的排班安排 | | |
| DOC-SCHEDULE-003 | 切换上一周排班 | 1. 在"我的排班"页面周视图 2. 点击"< 上一周"按钮 3. 观察排班网格 | 页面显示上一周的排班信息，日期列更新为上一周的日期范围 | | |
| DOC-SCHEDULE-004 | 切换下一周排班 | 1. 在"我的排班"页面周视图 2. 点击"> 下一周"按钮 3. 观察排班网格 | 页面显示下一周的排班信息，日期列更新为下一周的日期范围 | | |
| DOC-SCHEDULE-005 | 返回本周排班 | 1. 在"我的排班"页面周视图 2. 点击"本周"按钮 3. 观察排班网格 | 页面跳转回当前周的排班信息，日期列显示本周的日期范围 | | |
| DOC-SCHEDULE-006 | 查看排班统计信息 | 1. 在"我的排班"页面 2. 观察页面底部的统计卡片 | 页面底部正确显示"总排班数"和"可用排班"等统计信息 | | |
| DOC-SCHEDULE-007 | 刷新排班数据 | 1. 在"我的排班"页面 2. 点击"刷新"按钮 3. 观察排班数据 | 排班数据重新加载，显示最新的排班安排，显示"排班数据加载成功"提示 | | |

## 四、我的休假申请功能测试用例

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| DOC-LEAVE-001 | 申请休假 | 1. 进入"我的休假申请"页面 2. 点击"+ 申请休假"按钮 3. 在申请表单中填写假期类型、开始日期、结束日期、请假事由等信息 4. 上传请假证明（可选） 5. 点击"提交申请"按钮 | 系统成功创建休假申请，显示提交成功提示，新申请在列表中可见，状态为"待审批" | | |
| DOC-LEAVE-002 | 查看休假申请列表 | 1. 进入"我的休假申请"页面 2. 观察申请列表表格 | 表格正确显示所有休假申请，包含假期类型、开始日期、结束日期、请假事由、请假证明、审批状态、操作等信息 | | |
| DOC-LEAVE-003 | 撤销待审批的休假申请 | 1. 在"我的休假申请"页面 2. 找到状态为"待审批"的申请 3. 点击"撤销申请"按钮 4. 确认撤销操作 | 系统成功撤销休假申请，显示撤销成功提示，该申请从列表中移除或状态更新为已撤销 | | |
| DOC-LEAVE-004 | 查看请假证明 | 1. 在"我的休假申请"页面 2. 找到有"查看证明"链接的申请 3. 点击"查看证明"链接 | 系统在新窗口或对话框中打开请假证明文件，可以查看证明内容 | | |
| DOC-LEAVE-005 | 查看已批准/已拒绝的申请详情 | 1. 在"我的休假申请"页面 2. 找到状态为"已批准"或"已拒绝"的申请 3. 观察申请详情 | 申请列表显示完整的申请信息，包括审批状态、审批意见（如有）等信息 | | |

## 五、我的加号申请功能测试用例

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| DOC-SLOT-001 | 申请加号 | 1. 进入"我的加号申请"页面 2. 点击"+ 申请加号"按钮 3. 在申请表单中选择排班日期和时间段 4. 填写加号数量、患者信息、紧急程度、申请理由 5. 点击"提交申请"按钮 | 系统成功创建加号申请，显示提交成功提示，新申请在列表中可见，状态为"待审批" | | |
| DOC-SLOT-002 | 查看加号申请列表 | 1. 进入"我的加号申请"页面 2. 观察申请列表表格 | 表格正确显示所有加号申请，包含排班日期、时段、加号数量、紧急程度、申请理由、审批状态、操作等信息 | | |
| DOC-SLOT-003 | 查看加号申请详情 | 1. 在"我的加号申请"页面 2. 找到任意一条申请 3. 点击"查看详情"按钮 4. 查看详情对话框 | 系统弹出详情对话框，显示该申请的完整信息，包括排班信息、患者信息、申请理由、审批状态、审批意见（如有）等 | | |
| DOC-SLOT-004 | 查看不同状态的加号申请 | 1. 在"我的加号申请"页面 2. 观察列表中不同审批状态的申请 | "已通过"状态显示绿色标签，"已拒绝"状态显示红色标签，状态标识清晰明确 | | |

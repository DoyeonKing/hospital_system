## 常见问题处理

### 问题1：无法连接数据库
- [ ] 检查MySQL服务是否运行
- [ ] 检查端口3306是否被占用
- [ ] 检查用户名密码是否正确
- [ ] 检查防火墙设置

### 问题2：Postman请求失败
- [ ] 检查Spring Boot服务是否运行
- [ ] 检查端口8080是否被占用
- [ ] 检查请求URL是否正确
- [ ] 检查请求头是否正确

### 问题3：日志文件找不到
- [ ] 检查控制台输出
- [ ] 检查logs目录（如果配置了）
- [ ] 检查application.yml中的日志配置

### 问题4：资源占用测试困难
- [ ] 可以使用任务管理器简单记录
- [ ] 可以只记录初始和峰值
- [ ] 可以跳过长时间监控

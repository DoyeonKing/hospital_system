# 3.2.1 管理员权限界面显示测试用例

## 测试用例表格

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| ADMIN-UI-001 | 超级管理员界面显示 | 1. 使用admin_id=1的账号登录系统 2. 进入管理员后台首页 3. 观察页面显示的功能卡片 | 应显示全部11个功能卡片：数据大屏、科室信息管理、用户账户管理、管理员管理、排班管理、号别管理、就医规范管理、医生工时统计、患者签到、请假审批、加号审批、现场服务 | | |
| ADMIN-UI-002 | 患者管理员界面显示 | 1. 使用admin_id=2的账号登录系统 2. 进入管理员后台首页 3. 观察页面显示的功能卡片 | 应显示4个功能卡片：数据大屏、患者签到、加号审批、现场服务。不应显示：科室信息管理、用户账户管理、管理员管理、排班管理、号别管理、就医规范管理、医生工时统计、请假审批 | | |
| ADMIN-UI-003 | 权限验证-直接访问受限页面 | 1. 使用患者管理员(admin_id=2)账号登录 2. 在浏览器地址栏直接输入科室管理页面URL(/departments) 3. 观察页面响应 | 系统应阻止访问，可能的行为：重定向到首页、显示403禁止访问错误、或显示无权限提示。不应显示科室管理页面内容 | | |

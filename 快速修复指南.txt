真机调试网络连接问题 - 快速修复指南
========================================

问题：登录请求失败，连接被拒绝

已检测到的IP地址：
1. 172.20.10.3 (WiFi连接)
2. 192.168.137.1 (手机热点)
3. 26.206.1.21 (虚拟网卡)

解决方案（按顺序尝试）：

【方案1】修改IP地址
-------------------
1. 打开 uni_app/config/index.js
2. 找到 LOCAL_IP = '192.168.137.1'
3. 改为 LOCAL_IP = '172.20.10.3'
4. 保存，重新运行APP

【方案2】检查防火墙
-------------------
1. 控制面板 → Windows Defender 防火墙
2. 临时关闭防火墙（仅测试）
3. 测试登录功能
4. 如果成功，说明是防火墙问题，添加8080端口规则

【方案3】使用手机热点
-------------------
1. 手机开启热点
2. 电脑连接手机热点
3. 运行 ipconfig，查看新的IP地址
4. 修改 config/index.js 中的IP
5. 重新运行APP

【方案4】使用USB连接
-------------------
1. 用USB线连接手机
2. HBuilderX → 运行 → 运行到手机 → USB设备
3. 这样不需要配置IP

【方案5】检查网络】
-------------------
1. 确保手机和电脑在同一WiFi
2. 在手机浏览器访问：http://你的IP:8080
3. 如果无法访问，说明网络不通

测试命令：
---------
在手机浏览器访问：
http://172.20.10.3:8080
或
http://192.168.137.1:8080

如果能看到页面（即使报错），说明网络通了！

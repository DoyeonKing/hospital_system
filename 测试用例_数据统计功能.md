# 3.2.5 数据统计功能测试用例

## 测试用例表格

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| STATS-001 | 查看运营总览数据 | 1. 以管理员身份登录系统 2. 进入"数据大屏"页面 3. 默认显示"运营总览"标签页 4. 观察页面显示的KPI数据和图表 | 页面正确显示今日挂号量、今日出诊医生、当前候诊人数、累计注册用户等KPI数据，显示全院挂号趋势折线图和支付状态分布饼图 | | |
| STATS-002 | 切换标签页-医生资源分析 | 1. 在数据大屏页面 2. 点击"医生资源分析"标签页 3. 观察页面显示的数据和图表 | 页面切换到医生资源分析视图，正确显示医生总数、今日请假人数、科室总数等KPI数据，显示职称分布饼图和科室繁忙度Top 5柱状图 | | |
| STATS-003 | 切换标签页-患者群体画像 | 1. 在数据大屏页面 2. 点击"患者群体画像"标签页 3. 观察页面显示的数据和图表 | 页面切换到患者群体画像视图，正确显示本月新增注册、师生比例系数、累计爽约次数等KPI数据，显示用户增长趋势折线图、患者类型构成饼图和就诊时段热力图 | | |
| STATS-004 | 导出报表 | 1. 在数据大屏页面任意标签页 2. 点击"导出报表"按钮 3. 等待导出完成 4. 检查下载的文件 | 系统显示"正在生成报表"提示，完成后显示"报表导出成功"，浏览器自动下载PDF格式报表文件，文件包含当前标签页的所有数据图表 | | |
| STATS-005 | 退出大屏 | 1. 在数据大屏页面 2. 点击"退出大屏"按钮 | 系统返回到管理员后台首页，不再显示数据大屏页面 | | |

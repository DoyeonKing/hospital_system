# 3.2.3 排班管理功能测试用例

## 测试用例表格

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| SCHED-001 | 自动排班配置与生成 | 1. 进入"排班管理"->"自动排班"页面 2. 选择科室（父科室和子科室） 3. 设置排班时间范围（开始日期和结束日期） 4. 配置排班规则（每时段医生数、默认号源数、默认挂号费等） 5. 点击"生成并保存"按钮 | 系统成功生成排班，显示成功提示消息，排班数据保存到数据库，在排班看板中可以看到生成的排班 | | |
| SCHED-002 | 手动排班-拖拽排班 | 1. 进入"排班管理"->"排班看板"页面 2. 选择目标科室 3. 从底部"待排班医生"区域拖拽医生卡片到指定日期和时间段 4. 从底部"时间段卡片"区域拖拽时间段到已放置的医生卡片上 5. 从底部"可用办公地点"区域拖拽地点到医生卡片上 6. 观察保存提示消息 | 系统成功保存排班信息，显示"排班保存成功"提示，包含医生、日期、时间段、地点等信息，排班卡片正确显示在日历网格中 | | |
| SCHED-003 | 查看排班-视图切换 | 1. 进入排班看板页面 2. 点击"日视图"按钮 3. 点击"周视图"按钮 4. 点击"月视图"按钮 5. 观察不同视图下的排班显示 | 日视图显示单日详细排班信息，周视图显示一周的排班信息，月视图显示整月的排班概览，每个视图正确显示对应时间范围的排班数据 | | |
| SCHED-004 | 删除排班 | 1. 进入排班看板页面 2. 找到已存在的排班卡片 3. 点击排班卡片右上角的红色"X"按钮 | 排班卡片从日历网格中移除，系统显示删除成功提示消息，数据库中对应的排班记录被删除 | | |

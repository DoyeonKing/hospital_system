# 3.2.4 科室管理功能测试用例

## 测试用例表格

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| DEPT-001 | 创建新科室 | 1. 进入"科室信息管理"->"创建新科室"页面 2. 填写科室名称（如：心血管科）、选择父科室（如：内科）、填写科室描述 3. 点击"立即创建"按钮 | 系统成功创建科室，显示成功提示，新科室在科室列表中可见 | | |
| DEPT-002 | 编辑科室描述 | 1. 进入"科室信息管理"页面 2. 找到目标科室，点击"编辑"按钮 3. 修改科室描述信息 4. 点击"确认修改"按钮 | 系统成功更新科室描述，显示成功提示，修改后的描述在列表中正确显示 | | |
| DEPT-003 | 删除科室 | 1. 进入"科室信息管理"页面 2. 找到目标科室，点击"删除"按钮 3. 在确认对话框中点击"确定" | 系统成功删除科室，显示成功提示，该科室从列表中移除，科室下的医生和地点自动处理 | | |
| DEPT-004 | 添加症状-科室映射 | 1. 进入"症状-科室映射管理"页面 2. 点击"+ 添加映射"按钮 3. 填写症状关键词（如：头痛,发热）、选择推荐科室（如：神经科）、设置优先级（如：1） 4. 点击"确认添加"按钮 | 系统成功创建映射，显示成功提示，新映射在列表中可见 | | |
| DEPT-005 | 编辑症状-科室映射 | 1. 进入"症状-科室映射管理"页面 2. 找到目标映射，点击"编辑"按钮 3. 修改症状关键词或推荐科室 4. 点击"确认保存"按钮 | 系统成功更新映射信息，显示成功提示，修改后的信息在列表中正确显示 | | |
| DEPT-006 | 删除症状-科室映射 | 1. 进入"症状-科室映射管理"页面 2. 找到目标映射，点击"删除"按钮 3. 确认删除操作 | 系统成功删除映射，显示成功提示，该映射从列表中移除 | | |
| DEPT-007 | 搜索症状-科室映射 | 1. 进入"症状-科室映射管理"页面 2. 在"科室"下拉框中选择一个科室（如：神经科） 3. 点击"搜索"按钮 4. 观察列表显示 5. 点击"重置"按钮 | 列表只显示所选科室的映射，点击重置后列表恢复显示所有映射 | | |

## 🚀 现在立即测试（3步搞定）

### 步骤 1：启动项目

**如果你使用 HBuilderX：**
1. 打开 HBuilderX
2. 文件 → 打开目录 → 选择 `E:\hospital_system\uni_app`
3. 运行 → 运行到小程序模拟器 → 微信开发者工具
4. 等待自动编译完成

**如果你使用命令行：**
```bash
cd E:\hospital_system\uni_app
npm run dev:mp-weixin
```
然后手动打开微信开发者工具，导入项目：`E:\hospital_system\uni_app\unpackage\dist\dev\mp-weixin`

### 步骤 2：访问地图示例页面（最简单）

在微信开发者工具中：

1. 点击右上角"详情"按钮
2. 在"页面路径"输入框中输入：`pages/navigation/MapExample`
3. 点击"编译"按钮
4. 页面自动跳转到地图示例页面

### 步骤 3：测试功能

你会看到一个地图页面，有3个按钮：

1. **点击"📍 获取当前位置"按钮**
   - ✅ 应该弹出位置权限请求（第一次）
   - ✅ 点击"允许"
   - ✅ 地图中心移动到你的位置
   - ✅ 显示坐标信息

2. **测试地图交互**
   - ✅ 拖动地图（单指）
   - ✅ 缩放地图（双指）

3. **测试其他按钮**
   - ✅ 点击"重置地图"
   - ✅ 点击"添加标记"

## 🐛 如果遇到问题

### 问题 1：地图不显示
- 检查控制台错误
- 确认高德 Key 配置正确
- 刷新页面

### 问题 2：定位失败
- 微信开发者工具：设置 → 安全 → 开启定位权限
- 或在手机上测试（点击"预览"用微信扫码）

### 问题 3：找不到页面
- 确认页面路径正确
- 检查编译是否成功

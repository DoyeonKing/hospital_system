# 3.2.2 用户管理功能测试用例

## 测试用例表格

| 用例编号 | 测试用例名称 | 测试步骤 | 预期结果 | 实际结果 | 通过/失败 |
|---------|------------|---------|---------|---------|----------|
| USER-001 | 创建新患者账户 | 1. 以管理员身份登录系统 2. 进入"用户账户管理"->"创建新用户账户"页面 3. 填写必填字段：学号/工号(2021001001)、姓名(张三)、密码(123456)、角色(患者)、患者类型(学生)、身份证号(110101200001010011)、手机号(13810011001) 4. 可选填写过敏史和既往病史 5. 点击"立即创建"按钮 | 系统成功创建患者账户，显示成功提示消息，新创建的用户可以在用户列表中查看到 | | |
| USER-002 | 编辑患者信息 | 1. 以管理员身份登录系统 2. 进入"用户账户管理"->"编辑用户信息"页面 3. 选择"患者"标签页 4. 在搜索框中输入学号或姓名搜索患者 5. 点击目标患者的"编辑"按钮 6. 在编辑对话框中修改患者信息（如姓名、手机号、患者类型、状态等） 7. 如需修改密码，在"重置密码"字段输入新密码 8. 点击"保存"按钮 | 系统成功更新患者信息，显示成功提示消息，修改后的信息在列表中正确显示 | | |
| USER-003 | 患者档案管理 | 1. 以管理员身份登录系统 2. 进入"用户账户管理"->"患者档案管理"页面 3. 在列表中查看患者的过敏史、既往病史、黑名单状态 4. 修改某个患者的过敏史信息 5. 修改某个患者的既往病史信息 6. 切换某个患者的黑名单状态开关 7. 点击该患者的"保存"按钮 | 系统成功保存患者档案信息，显示成功提示消息，修改后的档案信息正确保存并在列表中显示 | | |

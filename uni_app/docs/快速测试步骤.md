# 🚀 快速测试步骤（5分钟上手）

## 第一步：启动项目

### 方式 A：使用 HBuilderX（最简单）

1. **打开 HBuilderX**
2. **文件 → 打开目录** → 选择 `E:\hospital_system\uni_app`
3. **运行 → 运行到小程序模拟器 → 微信开发者工具**
4. 等待编译完成，微信开发者工具会自动打开

### 方式 B：使用命令行

```bash
# 1. 打开终端（PowerShell 或 CMD）
cd E:\hospital_system\uni_app

# 2. 安装依赖（如果还没安装）
npm install

# 3. 编译微信小程序
npm run dev:mp-weixin

# 4. 手动打开微信开发者工具
# 导入项目：E:\hospital_system\uni_app\unpackage\dist\dev\mp-weixin
```

---

## 第二步：访问地图示例页面（最简单）

在微信开发者工具中：

1. **点击右上角的"详情"按钮**
2. **找到"页面路径"输入框**
3. **输入**：`pages/navigation/MapExample`
4. **点击"编译"按钮**
5. **页面会自动跳转到地图示例页面**

---

## 第三步：测试地图功能

进入地图示例页面后，你会看到：

- 📱 **地图显示区域**（上方）
- 🎛️ **控制按钮**（中间）：
  - 📍 获取当前位置
  - 🗺️ 重置地图  
  - ➕ 添加标记
- 📊 **位置信息显示**（下方）

### 测试步骤：

1. **点击"📍 获取当前位置"按钮**
   - ✅ 应该弹出"定位中..."的提示
   - ✅ 如果第一次使用，会弹出位置权限请求
   - ✅ 点击"允许"

2. **查看结果**
   - ✅ 地图中心会移动到你的位置（或默认位置）
   - ✅ 显示位置坐标信息（纬度、经度）
   - ✅ 地图上显示标记点

3. **测试地图交互**
   - ✅ 拖动地图（单指）
   - ✅ 缩放地图（双指）

4. **测试其他按钮**
   - ✅ 点击"重置地图" - 地图回到默认位置
   - ✅ 点击"添加标记" - 在地图中心添加标记点

## ⚠️ 常见问题

### Q1: 地图不显示？

**解决：**
- 检查控制台是否有错误
- 确认高德地图 Key 已配置（`manifest.json`）
- 尝试刷新页面

### Q2: 定位失败？

**解决：**
- **微信开发者工具**：设置 → 安全 → 开启定位权限
- 或在手机上测试（真机预览）

### Q3: 页面找不到？

**解决：**
- 确认页面路径正确：`pages/navigation/MapExample`
- 检查编译是否成功
- 查看控制台错误信息

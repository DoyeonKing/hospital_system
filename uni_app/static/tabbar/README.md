# TabBar 图标说明

## 需要的图标文件

请在此目录下放置以下 8 个图标文件（PNG 格式，建议尺寸：81x81 像素）：

### 1. 首页图标
- `home.png` - 未选中状态（灰色）
- `home-active.png` - 选中状态（青色 #38A2AC）

### 2. 我的预约图标
- `appointment.png` - 未选中状态（灰色）
- `appointment-active.png` - 选中状态（青色 #38A2AC）

### 3. 消息图标
- `message.png` - 未选中状态（灰色）
- `message-active.png` - 选中状态（青色 #38A2AC）

### 4. 我的图标
- `profile.png` - 未选中状态（灰色）
- `profile-active.png` - 选中状态（青色 #38A2AC）

## 图标设计建议

### 颜色
- **未选中状态**：#999999（灰色）
- **选中状态**：#38A2AC（青色/Teal）

### 尺寸
- 推荐尺寸：81x81 像素（@1x）
- 或者：162x162 像素（@2x）
- 或者：243x243 像素（@3x）

### 风格
- 简洁的线条图标
- 与医疗主题相关
- 保持一致的设计风格

## 图标内容建议

1. **首页 (home)**：房子图标
2. **我的预约 (appointment)**：日历或预约单图标
3. **消息 (message)**：消息气泡或铃铛图标
4. **我的 (profile)**：用户头像轮廓图标

## 在线图标资源

您可以从以下网站下载或设计图标：
- iconfont (阿里巴巴矢量图标库)
- iconpark (字节跳动图标库)
- flaticon
- iconfinder

## 临时方案

如果暂时没有图标，可以：
1. 使用纯色方块作为占位符
2. 或者暂时注释掉 pages.json 中的 iconPath 配置

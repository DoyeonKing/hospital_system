# TabBar 图标快速获取指南

## 🚀 最快速的方法（推荐）

### 方法一：使用 iconfont（阿里巴巴图标库）

1. **访问网站**：https://www.iconfont.cn/

2. **搜索并下载图标**：
   - 搜索 "home" → 选择一个喜欢的首页图标
   - 搜索 "calendar" → 选择一个日历/预约图标
   - 搜索 "message" → 选择一个消息图标
   - 搜索 "user" → 选择一个用户图标

3. **下载设置**：
   - 点击图标后选择"下载"
   - 格式：PNG
   - 尺寸：81px 或 100px
   - 颜色：
     * 第一次下载设置为 #999999（灰色）
     * 第二次下载设置为 #38A2AC（青色）

4. **重命名文件**：
   - 灰色版本命名为：`home.png`, `appointment.png`, `message.png`, `profile.png`
   - 青色版本命名为：`home-active.png`, `appointment-active.png`, `message-active.png`, `profile-active.png`

5. **放置文件**：
   将所有 8 个文件放到当前目录（`static/tabbar/`）

### 方法二：使用 IconPark（字节跳动图标库）

1. **访问网站**：https://iconpark.oceanengine.com/

2. **操作步骤**：
   - 搜索对应的图标关键词
   - 调整颜色和大小
   - 下载 PNG 格式
   - 按照上面的命名规则重命名

### 方法三：使用 Flaticon

1. **访问网站**：https://www.flaticon.com/

2. **下载免费图标**（需要注册）

## 📋 需要的 8 个文件清单

```
✅ home.png              (首页 - 灰色 #999999)
✅ home-active.png       (首页 - 青色 #38A2AC)
✅ appointment.png       (预约 - 灰色 #999999)
✅ appointment-active.png (预约 - 青色 #38A2AC)
✅ message.png           (消息 - 灰色 #999999)
✅ message-active.png    (消息 - 青色 #38A2AC)
✅ profile.png           (我的 - 灰色 #999999)
✅ profile-active.png    (我的 - 青色 #38A2AC)
```

## 🎨 图标规格要求

- **格式**：PNG（支持透明背景）
- **尺寸**：81x81 像素（推荐）或 100x100 像素
- **颜色**：
  - 未选中：#999999（灰色）
  - 选中：#38A2AC（青色/Teal）
- **背景**：透明

## ⚡ 完成后

1. 确保 8 个文件都已放置在 `static/tabbar/` 目录
2. 重新编译项目（HBuilderX 中点击"运行"）
3. 底部导航栏将显示图标

## 💡 临时解决方案

如果暂时无法获取图标，可以：
1. 使用任意 PNG 图片作为占位符（调整为 81x81 像素）
2. 或者在 `pages.json` 中暂时注释掉 `iconPath` 和 `selectedIconPath`

## 🔧 验证图标是否正确

文件放置后，目录结构应该是：
```
static/
  └── tabbar/
      ├── home.png
      ├── home-active.png
      ├── appointment.png
      ├── appointment-active.png
      ├── message.png
      ├── message-active.png
      ├── profile.png
      └── profile-active.png
```

## ❓ 常见问题

**Q: 图标不显示？**
A: 检查文件名是否完全匹配（区分大小写），路径是否正确

**Q: 图标显示模糊？**
A: 使用更高分辨率的图标（建议 162x162 或 243x243）

**Q: 可以使用 SVG 吗？**
A: uni-app 的 tabBar 不支持 SVG，必须使用 PNG

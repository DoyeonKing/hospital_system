# 华为云API网关共享版使用指南

## 📍 当前情况说明

你看到的"实例管理"页面显示"无记录"，这是正常的。华为云API网关有两种版本：

1. **共享版**：免费，无需创建实例，直接使用
2. **专享版**：付费，需要先创建实例

**你看到"购买实例"按钮是针对专享版的，共享版不需要！**

---

## 🎯 解决方案：使用共享版（免费）

### 方法一：直接创建API分组（推荐）

共享版不需要先创建实例，可以直接创建API分组：

1. **在左侧菜单找到"API分组"**
   - 如果左侧菜单没有展开，点击左侧的"APIG"
   - 查找 **"API分组"** 或 **"API管理"** → **"API分组"**

2. **如果找不到"API分组"菜单**
   - 尝试在顶部搜索栏输入：**API分组**
   - 或直接访问URL（将当前URL中的路径改为）：
     ```
     https://console.huaweicloud.com/apig2/?locale=zh-cn&region=cn-north-4#/apig2/manage/apiGroups
     ```

3. **创建分组**
   - 点击 **"创建分组"** 或 **"新建分组"**
   - 填写分组信息（共享版会自动选择，无需手动选择实例）
   - 创建后会自动分配测试域名

### 方法二：切换到共享版视图

1. **检查左侧菜单**
   - 查看是否有 **"共享版"** 或 **"共享实例"** 选项
   - 如果有，点击进入

2. **或者直接访问共享版**
   - 在浏览器地址栏，尝试访问：
     ```
     https://console.huaweicloud.com/apig2/?locale=zh-cn&region=cn-north-4#/apig2/shared
     ```

## 🚀 快速操作步骤（共享版）

### 步骤1：进入API分组页面

**方式A：通过菜单**
1. 在左侧菜单找到 **"API分组"**
2. 点击进入

**方式B：通过搜索**
1. 在顶部搜索栏输入：**API分组**
2. 点击搜索结果

**方式C：直接访问**
1. 复制以下URL到浏览器：
   ```
   https://console.huaweicloud.com/apig2/?locale=zh-cn&region=cn-north-4#/apig2/manage/apiGroups
   ```
2. 替换 `region=cn-north-4` 为你的实际区域

### 步骤2：创建API分组

1. 点击 **"创建分组"** 或 **"新建分组"** 按钮
2. 填写信息：
   ```
   分组名称：hospital-api
   描述：医院系统API（可选）
   ```
3. 点击 **"确定"** 或 **"创建"**

### 步骤3：获取测试域名

创建分组后：
- 系统会自动分配测试域名
- 格式：`https://xxxxx.apigw.huaweicloud.com`
- 在分组详情页面可以看到
- **这就是你要用的免费域名！**

### 步骤4：创建API

1. 在分组页面点击 **"创建API"**
2. 配置API：
   ```
   API名称：医院系统API
   请求路径：/{path}
   请求方法：GET、POST、PUT、DELETE
   后端服务类型：HTTP
   后端服务地址：http://你的EIP:8080
   后端请求路径：/{path}
   ```
3. 点击 **"发布"**

## 📝 如果还是找不到"API分组"

### 可能的原因：

1. **权限问题**
   - 确认账号有API网关的使用权限
   - 联系管理员授权

2. **区域问题**
   - 确认在正确的区域
   - 尝试切换到其他区域

3. **界面版本问题**
   - 尝试刷新页面（F5）
   - 清除浏览器缓存
   - 尝试使用其他浏览器

4. **服务未开通**
   - 可能需要先开通API网关服务
   - 查看是否有"开通服务"的提示

### 解决方法：

1. **查看左侧完整菜单**
   - 确认左侧菜单完全展开
   - 查找所有包含"API"的菜单项

2. **使用顶部搜索**
   - 搜索：**API分组**
   - 搜索：**创建分组**
   - 搜索：**API管理**

3. **查看帮助文档**
   - 点击页面上的"使用指南"或"帮助"
   - 查看华为云官方文档

## 🔧 如果确实需要创建实例

如果你确定需要使用专享版（付费），可以：

1. 点击 **"购买实例"** 按钮
2. 选择配置和时长
3. 完成支付
4. 然后创建API分组

**但建议先尝试共享版**，因为：
- 免费
- 适合测试
- 功能足够体验版使用

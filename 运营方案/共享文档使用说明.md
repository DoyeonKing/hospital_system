# 共享文档使用说明

## 📋 文档说明

`04_共享文档模板.md` 是我们小组运营方案的共享文档模板。**所有成员的内容最终都要写在一个文档里**，最终提交一份完整的Word文档。

---

## 🎯 使用方式

### 方式一：使用在线协作工具（推荐）

#### 推荐工具：
1. **腾讯文档**（推荐）
   - 网址：https://docs.qq.com
   - 优点：免费、支持多人同时编辑、实时同步、可导出Word
   - 操作：创建一个在线文档，所有成员共享编辑权限

2. **石墨文档**
   - 网址：https://shimo.im
   - 优点：界面美观、支持多人协作、可导出Word

3. **金山文档**
   - 网址：https://www.kdocs.cn
   - 优点：与WPS兼容性好、支持多人协作

#### 操作步骤：
1. 选择一位成员（建议成员D）创建在线文档
2. 将 `04_共享文档模板.md` 的内容复制到在线文档中
3. 分享文档链接给所有成员，设置编辑权限
4. 各成员在自己负责的章节填写内容
5. 完成后导出为Word文档（.docx格式）

---

### 方式二：使用Word文档协作

#### 操作步骤：
1. 将 `04_共享文档模板.md` 的内容复制到Word文档中
2. 保存为Word格式（.docx）
3. 使用Word的"修订"功能或"批注"功能进行协作
4. 或使用OneDrive/Google Drive等云盘共享文档

#### Word协作功能：
- **修订模式**：可以追踪每个人的修改
- **批注功能**：可以添加评论和建议
- **保护文档**：可以设置每个人只能编辑指定区域

---

### 方式三：使用Markdown协作后转换

#### 操作步骤：
1. 所有成员在Markdown文件中填写内容
2. 使用工具将Markdown转换为Word：
   - 在线工具：https://www.markdowntoword.com
   - VS Code插件：Markdown PDF
   - Pandoc命令行工具

## 📝 填写说明

### 1. 各成员负责的章节

| 成员 | 负责章节 | 说明 |
|------|---------|------|
| 成员A | 第2章：市场调研与用户分析 | 所有子章节都需要填写 |
| 成员B | 第3章：运营策略方案<br>第4章：推广方案 | 所有子章节都需要填写 |
| 成员C | 第5章：产品优化方案 | 所有子章节都需要填写 |
| 成员D | 第1章：产品概要介绍（整合）<br>第6章：实施计划与预算<br>第7章：效果评估<br>第8章：总结<br>附录：成员分工说明 | 负责整合和撰写 |

### 2. 填写要求

#### ✅ 必须填写的内容：
- 所有标注了"[成员X请填写]"的位置
- 表格中的所有单元格
- 示例中的具体数据（如目标值、预算金额等）

#### 📝 填写建议：
- **内容要详细**：不要只写一句话，尽量展开说明
- **数据要具体**：尽量填写具体的数字，不要用"X"、"待填写"等占位符
- **逻辑要清晰**：每个方案都要有清晰的逻辑和理由
- **格式要统一**：保持与模板一致的格式风格

### 3. 标记完成状态

建议在完成自己负责的章节后，在章节标题后添加标记：
- ✅ 已完成
- 🔄 进行中
- ⏳ 未开始

例如：
```
## 2. 市场调研与用户分析 ✅ 已完成
```

## ⚠️ 注意事项

### 1. 文档格式要求
- **文件格式**：最终提交必须是Word文档（.docx）
- **页码**：需要添加页码
- **目录**：需要自动生成目录
- **页眉页脚**：可以添加页眉页脚（如项目名称、日期等）

### 2. 内容要求
- **必须包含成员分工说明**：在附录中必须明确说明各成员的工作量百分比
- **内容要完整**：所有章节都要有实质性内容，不能只是框架
- **逻辑要连贯**：各章节之间的内容要有逻辑关联

### 3. 协作注意事项
- **不要删除他人的内容**：只能编辑自己负责的章节
- **及时保存**：编辑后及时保存，避免内容丢失
- **定期同步**：建议每天结束时同步一次文档
- **标注清楚**：如果有疑问或需要讨论，使用批注功能标注

## 💡 常见问题

### Q1: 如果某个章节写不完怎么办？
**A**: 先完成核心内容，后续可以在Week 5-6期间继续完善。但要保证核心框架和主要内容完整。

### Q2: 可以在章节中添加图表吗？
**A**: 可以！建议添加图表来更好地展示内容，如：
- 用户画像图表
- 竞品对比表格
- 运营策略流程图
- 预算分配饼图
- 等等

### Q3: 格式不统一怎么办？
**A**: 成员D在整合文档时会统一格式。但建议大家在填写时就尽量保持一致，减少后期工作量。

### Q4: 可以在线文档和本地文档同时使用吗？
**A**: 不建议！建议统一使用一种方式，避免版本混乱。推荐使用在线协作工具。

### Q5: 如何导出Word文档？
**A**: 
- 腾讯文档：右上角"导出" → "Word"
- 石墨文档：右上角"导出" → "Word"
- 金山文档：文件 → 另存为 → Word格式

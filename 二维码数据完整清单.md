## 📝 二维码生成方式

### 快速生成所有二维码（推荐）

**方法1：使用HTML工具**
- 打开 `tools/qrcode_generator.html`
- 会自动显示所有14个节点的二维码
- 点击"下载"按钮保存

**方法2：使用Python脚本**
```bash
cd tools
python generate_all_qrcodes.py
```

**方法3：使用在线API**
直接访问以下链接生成（替换节点ID 1-14）：
```
https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=HOSPITAL_NODE_1
https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=HOSPITAL_NODE_2
... 以此类推到 HOSPITAL_NODE_14
```
